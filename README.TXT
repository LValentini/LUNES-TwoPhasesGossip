This is a simulation model for the ARTÌS RTI that implements the Large
Unstructured NEtwork Simulator (LUNES). In this software distribution is
provided: all the source code of the simulation model, the shell scripts
and adjunctive software used for performance evaluation and some other 
scripts that can be obtained for a complete performance evaluation.

For details about the implementation please see the file: mig-agents.c
it provides the logical structure of the simulation model implementation 
and many other useful details.

For more information on how to use the simulator please see USAGE.TXT
